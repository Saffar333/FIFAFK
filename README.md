# ЧМ-2026 — Турнир Костя vs Фаргат

Сайт-калькулятор очков с **автоматическим подтягиванием результатов** ЧМ-2026. Считает очки по вашим правилам в реальном времени, подсвечивает незавершённые группы жёлтым, определяет лидера и тай-брейк.

## Как это работает

- `index.html` — сам сайт (вся логика подсчёта, интерфейс).
- `api/standings.js` — бесплатная serverless-функция Vercel. Она на сервере забирает данные с публичного API ESPN (без ключа, без CORS), сопоставляет ваши 16 команд и вычисляет для каждой: стадию, 1-е место в группе и статус (идёт группа / в плей-офф / вылет / чемпион).
- Сайт раз в 2 минуты опрашивает `/api/standings` и обновляет счёт сам. Ничего вручную вносить не нужно.
- Любую команду можно **поправить вручную** — тогда авто-обновление её не трогает, пока не нажать «↻ авто» в строке (или «Сбросить ручные правки» внизу).

Всё работает на бесплатном тарифе Vercel (Hobby). Отдельный сервер и платные сервисы не нужны.

## Деплой на Vercel (любой из способов)

**Способ 1 — перетащить папку (проще всего)**
1. Зайти на vercel.com → Add New → Project.
2. Перетащить эту папку (с `index.html`, `api/`, `package.json`).
3. Deploy. Готово — Vercel сам поймёт, что `api/standings.js` это функция, а `index.html` — главная страница.

**Способ 2 — через CLI**
```
npm i -g vercel
cd <эта-папка>
vercel        # первый деплой
vercel --prod # боевой
```

**Способ 3 — через GitHub**
1. Залить папку в репозиторий.
2. На Vercel: Add New → Project → импортировать репозиторий → Deploy.

Никаких настроек, переменных окружения или build-команд не требуется.

## Проверка

После деплоя открой `https://<твой-проект>.vercel.app/api/standings` — должен вернуться JSON со статусом команд. Если `tournamentStarted: false`, значит источник пока без свежих результатов (сайт в этом случае покажет предупреждение и оставит ручной ввод).

## Локальный запуск

Если открыть `index.html` просто файлом — подсчёт и ручной ввод работают, но `/api/standings` недоступен (нет сервера). Для локальной проверки функции: `vercel dev` в папке проекта.

## Если ESPN окажется неточным

Источник можно сменить, не трогая интерфейс — вся загрузка изолирована в `api/standings.js` (функция `derive()` принимает JSON и отдаёт статусы команд). Можно подключить, например, football-data.org по бесплатному ключу.
